# Adaptive Skill Tree Generator for Personalized Learning Paths

## Overview

Adaptive Skill Tree Generator is an AI-powered personalized learning platform that creates dynamic learning paths for students based on their skills, quiz performance, and learning objectives.

The system analyzes a student's strengths and weaknesses through assessments and generates a customized skill tree with recommended topics and career guidance.

## Features

* Personalized learning path generation
* Skill assessment through quizzes
* Adaptive skill tree creation
* Career guidance recommendations
* Progress visualization using graphs
* Admin dashboard for managing questions and subjects
* Student dashboard for learning and tracking progress

## Tech Stack

* Python
* Django
* HTML, CSS, JavaScript
* SQLite Database
* Machine Learning concepts for recommendation generation

## Project Structure

* AdminApp – Admin functionalities
* StudentApp – Student functionalities
* Dataset – Quiz dataset
* Templates – Frontend pages
* Static – CSS, JS, and images

## Installation

```bash
git clone <repository-url>
cd Adaptive_SkillTree
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
